Syntax highlighting in the editor plugins and themes

== Description ==

Plugin Syntax Highlighting Editor replaces a simple editor for editing files textarea themes or plugins in the admin panel. 
The new editor has syntax highlighting for your own language. It is very convenient to read code than plain text. 
Next available syntaxes: 
html 
css 
js 
php 
xml 
sql 

In a plugin, you can specify the following settings: 
The choice of syntactic default editor 
Select language Default Editor 

p.s. Sorry, this text is translated G Translator

== Installation ==

1. Download the plugin Syntax Highlighting Editor 
2. Pumped into the folder wp-content/plugins / 
3. Activate the plugin 
4. Look in the Options section, click on the SH Editor.
5. Trying to edit the plugin installed or themes.
